## Turing T-SHIRTSHOP Backend

## ENV
For production you should update the .env 
For development it is static with my amazon rds endpoint can be updated at src/database/connection.js 
```bash
DB_USER=
DB_PASS=
DB_DATABASE=
DB_HOST=
DB_PORT=
```

----
## Installation
```bash
$ yarn 
```
## Run the project
```bash
$ yarn  start
```

## Generate production files
```bash
$ yarn build
```
